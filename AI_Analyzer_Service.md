# Step 2 AI Analyzer Service

This service provides capabilities for:

## Field Extraction
- Automatically identifies and extracts relevant fields from input data.

## Intent Detection
- Analyzes user input to detect the intent behind the query or command.

## Confidence Scoring
- Provides a confidence score to indicate the reliability of the detected intent and extracted fields.

### Usage
The AI Analyzer service can be integrated into various applications to enhance data processing and user interaction capabilities.
